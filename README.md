# Shopify Featured Product Section

Seção customizada criada do zero para o tema Horizon do Shopify.

## O que foi feito
- Estrutura Liquid com integração dinâmica ao catálogo de produtos do Shopify
- Layout responsivo com grid CSS (imagem + conteúdo)
- Badge, eyebrow, descrição e preço puxados diretamente do produto selecionado
- Schema completo para edição pelo admin do tema sem tocar no código
- Compatível com o tema Horizon

## Tecnologias
Liquid · CSS · Shopify Schema
